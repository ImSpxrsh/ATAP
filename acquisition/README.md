## The one-paragraph pitch (why this is novel)
Prior ATAP work is solid-tumor and old. Nobody has (a) computationally identified the
BAX/BAK-null, venetoclax-resistant blood-cancer population as the rational ATAP target,
(b) posed executioner availability as a *spatial routing* question, or (c) tied it to
*functional* apoptotic priming and a single falsifiable bench test. Doing (a)+(b) is a
strong poster; adding functional priming (#1/#2) makes it a methods-and-biology paper;
adding (#4) makes it a breakthrough.
